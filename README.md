# Project Graphene

To predict Material Removal Rate(MRR) and Surface Roughness(RA) 
from FEED, RPM and DOC of a machine and Graphene percentage using
various regression algorithms. 

### Finished

* Decision tree
* Linear Regression
* Lasso Regression
* Ridge Regression

### Ongoing

* SVM
